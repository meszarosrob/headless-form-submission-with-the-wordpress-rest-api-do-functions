# Headless Form Submission With the WordPress REST API + DigitalOcean Functions

This repository provides the mocked APIs for the CodePen embeds used in the [Headless Form Submission With the WordPress REST API](https://css-tricks.com/headless-form-submission-with-the-wordpress-rest-api/) article.

The reason and the approach are detailed in the [Keeping Article Demos Alive When Third-Party APIs Die](https://css-tricks.com/keeping-article-demos-alive-when-third-party-apis-die/) follow-up article.

This repository is made public to be a reference material.
